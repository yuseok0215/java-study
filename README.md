## 진행 방식
1. 스터디 진도에 맞게 책을 읽고 각자 맡은 파트에 대해 궁금증이나 중요한 포인트를 issue로 생성한다.
2. 작성한 issue를 이용해서 맡은 파트에 전반적인 내용과 핵심을 설명한다.
3. 읽으면서 들었던 의문점이나 궁금증을 팀원들에게 공유하고 그에 대한 각자의 생각을 공유한다.
